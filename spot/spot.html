基本用法

        
        Messenger().post({
            message: text, //弹出框的内容 可以是参数
            type: 'error', //弹出框的形式：成功提示框 绿色（'success'）/报错提示框 红色（error）/ 消息弹出框 蓝色（info）            
        })
        Messenger().run();

消息框常用设置

        Messenger.options = {
            extraClasses: 'messenger-fixed messenger-on-top',//messenger-fixed 消息框为弹出层 不设置默认为右下/ messenger-on-top消息框弹出位置 不设置默认为右下
            theme: 'future', //弹出框风格  可选风格有air block flat ice future等等
            hideAfter: 10,//设置多少秒后消失
            showCloseButton: true//设置消息框关闭按钮
        }
